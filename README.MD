# Redis-all

本项目为Redis各类实战的案例合集，结合应用场景与案例实战，包含特性如下：

- 基于Redis+Token，整合验证码，实现登录注册、权限校验、Token续期
- 缓存穿透解决方案（回种空值）
- 缓存击穿解决方案（逻辑过期）
- 缓存雪崩解决方案（缓存雪崩）
- 缓存预热与缓存构建
